PIP
---

Installing modules:

	pip install [modulename]

Install module and upgrade if possible:

	pip install --upgrade [modulename]


VIRTUALENV
----------

Install:

	pip install virtualenv

Creating new virtualenv:

	virtualenv [envname]

Activating environment:

	[envname]/Scripts/activate (in Windows: [envname]\Scripts\activate)

	NOTE: if you are using Windows Powershell, this doesn't work as expected
		Fallback to using `cmd`. E.g. run `cmd` inside PowerShell

Freezing requirements from env:

	pip freeze > requirements.txt
